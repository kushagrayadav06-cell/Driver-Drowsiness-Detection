# AI Driver Drowsiness Detection System

## Overview

This project detects driver drowsiness in real time using MediaPipe Face Mesh and OpenCV.

## Features

- Real-time face monitoring
- Drowsiness detection using EAR
- Alarm sound alert
- Face-not-visible warning sound
- Continuous monitoring
- Python-based implementation

## Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy
- Pygame

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python drowsiness_mediapipe.py
```
